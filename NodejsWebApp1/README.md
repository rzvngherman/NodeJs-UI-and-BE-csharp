# NodejsWebApp1


ENDPOINTS:

1) GET '/books'
-> get all books

2) POST '/books'
-> insert a book

3) GET '/book?title=...'
-> get a book by title

4) DELETE '/book'
-> delete a book by title

5) GET '/authors'
-> get all authors

6) GET '/date?year=...&month=...'
-> display both year and month from query string
